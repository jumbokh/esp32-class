三、Arduino 接腳認識
1 Arduino UNO 控制板
　　Uno 使用的時間較久，許多書籍、文件、網路上的分享、以及程式庫，幾乎都以Uno 為基準，因此首次接觸微控制器或Arduino 的人，大多還是以Uno 板為主。
　　UNO控制板最大的特色是採用了另一個微控制器（MEGA16U2或MEGA8U2），作為Arduino 與電腦USB 溝通的序列轉換介面。
　　Arduino 的板子目前已經更新到了Arduino UNO R3 版本，它是一個以ATmega328為核心的控制板，它共有14 個數字I/O 接腳（其中六個可以做PWM 輸出），6 個類比模擬接腳，一個16MHz 晶體振盪器，一個USB 接頭，一個DC 電源插座，一個ICSP header 和一個重置按鈕。
