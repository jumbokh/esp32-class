一、Arduino的介紹
　據說一開始是因為Massimo Banzi 的學生們經常抱怨找不到便宜好用的微控制器，在2005 年冬天，Massimo Banzi 跟朋友David Cuartielles（西班牙籍晶片工程師）討論了這個問題。兩人最後決定設計自己的電路板，並引入了Banzi 的學生David Mellis 為電路板設計程式語言。兩天以後，David Mellis 就寫出了程式碼，又過了三天，電路板隨之完工了。這塊電路板被命名為Arduino。除了以上原因外，團隊還希望讓即使不懂電腦程式設計的人，也能用Arduino 做出很酷的東西，例如對感測器作出回應，讓燈光閃爍，還能控制馬達，讓機器手臂動作等等。[維基百科，Arduino]
　Arduino 是一塊基於創用CC[註二] 開放原始碼的I/O 介面板，使用者可以以 Arduino 為基礎，發展出各自需求的硬體，目前市面上已發展出許多不同需求版本的主控板供使用者選擇。
