一、認識溫度感測模組
Arduino 本身支援類比輸入接腳，可以用來讀取類比輸入0~5V 的電壓，再經由內部以10bits 的A/D 轉換器，轉換成解析度1024 的數位訊號，這讓我們可以很輕易的完成像TMP36（圖12-1）這種輸入類比溫度訊號的溫度感測器。
另外還有一些溫度感測器內部已經具有A/D 轉換功能，直接以串列傳輸方式將數位訊號傳送出來，例如TC77、DS1821（如圖12-2、12-3），我們只要接收它所送出來的串列資料，就可以知道溫度的大小了，這種傳輸方式因為接收到資料時已經是數位訊號了，因此不需使用Arduino 的類比輸入接腳，只要使用數位接腳就可以了。
在這裡我們將選擇低成本和尺寸小的TC77 來做為本單元溫度感測的元件，其也常應用在生活上的3C 產品溫度檢測用，如硬碟、筆電、智慧型手機內的過熱保護偵測等。
