一、認識紅外線模組
在本單元中，我們除將教你認識何謂紅外線外，還將教你如何使用我們生活中最常用，也最普遍的遙控器，究竟它發射出來的訊號格式如何？我們只要知道紅外線遙控器發射出來的訊號格式，就可以用它來控制家中的電器了，進而應用在我們的專題裡，現在我們就開始進入紅外線的世界吧！
1 何謂紅外線？
　　我們一般人的眼睛可看得到的電磁波波長大約在400 nm（紫）到700nm（紅）之間，一般稱為「可見光區域」；而紅外線（infrared、IR），一般俗稱的紅外光，他的波長大概在750nm（奈米）至1mm（毫米）之間，所以紅外線對我們人類來說是屬於看不見的不可見光，現今生活上的種種控制和通訊，如電器遙控、探測儀器、醫療、軍事等方面，都被廣泛的使用。
