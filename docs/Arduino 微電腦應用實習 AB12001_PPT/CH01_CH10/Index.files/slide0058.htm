三、Arduino 接腳認識
　現僅將OZONE 與Leonardo 不同的地方列出來說明，其他部分均與Leonardo相同：
1. USB 接頭：使用Mini USB 接頭，可使用USB 下載程式或除錯。
2. cmdBUS ™接腳：用來連接利基週邊模組，分別是VIN、GND、SDA、SCL、EVT、SYN，連接時請注意腳位對應，將Vin 對接OZONE 上的Vin 腳位，若是腳位錯誤可能造成模組燒毀。
3. ICSP 插座：ICSP 可使用SPI 程式庫來做SPI 傳輸（Serial Peripheral Interface），插座接腳有MOSI、MISO、SCK、RESET、GND、5V。
